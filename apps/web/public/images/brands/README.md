# Brand logos for Brand Showcase

Place **white or single-color** logo files here. They will be shown in monochrome white on the landing page.

**Filename (slug):**

| Brand            | Filename           |
|------------------|--------------------|
| Zara             | `zara.svg`         |
| Massimo Dutti    | `massimo-dutti.svg` |
| Supreme          | `supreme.svg`      |
| Stüssy           | `stussy.svg`       |
| Nike             | `nike.svg`         |
| Stone Island     | `stone-island.svg` |
| Prada            | `prada.svg`        |
| Adidas           | `adidas.svg`       |
| Uniqlo           | `uniqlo.svg`       |
| COS              | `cos.svg`          |
| Brandy Melville  | `brandy-melville.svg` |

- Use **SVG** (e.g. `zara.svg`) or **PNG** (e.g. `zara.png`) with transparent background. SVG is tried first.
- If both are missing, the brand name is shown as text.
